# ZWDS Star Interpretation Data

## Overview
This directory contains star interpretation data generated from the master's teachings for use in the Destiny Navigator's InsightsPanel component.

## Files
- `star-interpretations.ts` - Complete star data for all 18 stars (generated by agents)

## Data Structure

### StarData Interface
```typescript
interface StarData {
  name: string;              // English name
  chineseName: string;       // Chinese name
  category: "major" | "minor"; // Star classification
  attributes: {
    authority: number;       // 0-100: Leadership, command
    resources: number;       // 0-100: Wealth, assets
    strategy: number;        // 0-100: Planning, foresight
    discipline: number;      // 0-100: Execution, consistency
    flow: number;           // 0-100: Ease, momentum
  };
  keywords: string[];       // 5 actionable keywords
  essences: {
    life: string;           // 10-15 words per aspect
    siblings: string;
    relationships: string;
    children: string;
    wealth: string;
    health: string;
    travel: string;
    social: string;
    career: string;
    home: string;
    fortune: string;
    parents: string;
  };
}
```

## Usage

```typescript
import { STAR_INTERPRETATIONS } from './star-data/star-interpretations';

// Get star data
const ziWeiData = STAR_INTERPRETATIONS["紫微"];

// Access attributes
const authority = ziWeiData.attributes.authority; // 95

// Get essence for specific aspect
const wealthEssence = ziWeiData.essences.wealth;

// Get keywords
const keywords = ziWeiData.keywords; // ["authority", "leadership", ...]
```

## Content Guidelines

1. **Grounded in Master's Teachings** - All content derived from reference materials
2. **No Templates** - Each essence is unique prose
3. **Strategic Language** - Modern, actionable framing (not generic astrology)
4. **Comparative Attributes** - Scored relative to other stars
5. **10-15 Words** - Concise, compelling essences

## Generation Process

Data is generated sequentially by 3 agents:
1. **Agent 1:** Power & Authority stars (6)
2. **Agent 2:** Action & Transformation stars (6)
3. **Agent 3:** Harmony & Support stars (6)

Total: 18 stars × 12 aspects = 216 unique essences
